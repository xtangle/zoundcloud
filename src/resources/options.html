Adds ID3v2 metadata tags to all tracks downloaded that have the .mp3 extension. Metadata data is fetched from SoundCloud's API.

The added metadata includes: cover art, title, album artist, genres, duration, release year, bpm, artist url, audio source url, and description.

Expand for details.

All tracks will be downloaded in mp3 format. This ensures that metadata will always be added when the 'Add Metadata to mp3 files' option is enabled.

If this option is disabled, the original uploaded format of a track (often of higher quality format) will take precedence when downloading. However, metadata will not be added to non-mp3 files.

Before metadata is added and the file name is established, if the track title contains a substring matching the provided regular expression pattern, the matching part will be removed.

The default setting attempts to remove strings containing 'Free Download' or 'Download Link' patterns that appears in the end of popular track titles.

When entering a custom regular expression, make sure it conforms to Javascript's RegExp format. The provided regex string will be passed into a RegExp constructor with the global flag turned off and the case insensitive flag turned on.
